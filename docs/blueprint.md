# **App Name**: SceneCraft AI: Storyboarder

## Core Features:

- Scene Generation: AI-powered Scene Generation: Users input a text prompt, and the tool uses a generative AI model to create a storyboard frame representing the scene. The prompt incorporates directorial instructions for best results.
- Prompt Enhancement: Prompt Enrichment: Uses an LLM to take the user's prompt and enrich it with additional cinematic descriptors, ensuring consistency with other shots.
- Scene Ordering: Storyboard Sequencing: Drag and drop interface to rearrange the order of scenes within the storyboard.
- Annotations: Scene Description: Allow user to add descriptive notes or dialogue snippets for each scene in the storyboard
- Export to PDF: Export to PDF: Allow user to download completed storyboard as a PDF file

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082), evoking a sense of creativity and sophistication, fitting for a professional tool.
- Background color: Very light gray (#F0F0F0), providing a neutral backdrop that highlights the storyboard content. Desaturated hue of the primary, adjusted for brightness in a light scheme.
- Accent color: Violet (#8F00FF), a vibrant analogous color used to draw attention to key UI elements and calls to action.
- Body and headline font: 'Inter', a sans-serif font, for a modern, objective look that is readable for both headlines and body text.
- Minimalist outline icons. Use a consistent style throughout the application.
- Grid-based layout. Allows for clear content hierarchy and easy arrangement and re-arrangement.
- Subtle transitions. Add minor animations to smooth user flow (e.g., scene transitions). Ensure that the animations aren't distractive or irritating to the user.
